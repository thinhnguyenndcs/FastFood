# Website bán thức ăn nhanh
Sử dụng công nghệ HTML, CSS và Nodejs.

## Các bước chạy hệ thống

### > Đầu tiên chạy lệnh 
### `npm install`

### > Sau đó chạy lệnh
### `npm start`

Chạy ứng dụng ở chế độ development.\
Mở [http://localhost:3000](http://localhost:3000) để xem hệ thống trên trình duyệt.
